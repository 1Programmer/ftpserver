# ftpserver [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]
> Simple FTP server

## Installation

```sh
$ npm install --save ftpserver
```

## Usage

```js
var ftpserver = require('ftpserver');

ftpserver('Rainbow');
```
